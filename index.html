📤 Exporter les listes

Télécharge un fichier JSON avec toutes tes listes. Tu pourras le réimporter sur n'importe quel appareil ou navigateur.

📦 Importer des listes

Importe un fichier JSON exporté depuis VocabPro. Les listes existantes ayant le même nom seront remplacées, les autres seront ajoutées.

⚠️ Vérifie le fichier avant d'importer.

📋 Format du fichier JSON

Tu peux aussi créer tes listes dans un tableur (Excel/Sheets) et les exporter manuellement au format JSON suivant :

[
  {
    "name": "Chapitre 1 – Le corps",
    "filiere": "AEPA",
    "classe": "2AEPA",
    "words": [
      { "fr": "le bras", "en": "arm", "definition": "part of the body from shoulder to hand" },
      { "fr": "la jambe", "en": "leg", "definition": "" }
    ]
  }
]
